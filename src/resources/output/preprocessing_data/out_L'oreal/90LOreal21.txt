Titre: Rouge à lèvres, le "nude" selon L'Oréal Paris. Maquillage. Dans la gamme Color Riche arrivent les Accords naturels. Ces nouveaux rouges à lèvres (12,49 € , février) vont par paire : deux pour les blondes, les châtains, les brunes... car, outre leur qualité de confort et d'hydratation (omega 3 et vitamine E) comme Color Riche, les Accords Naturels s'inscrivent dans la tendance du naturel (qui sublime) et sont conçus pour s'harmoniser avec les différentes carnations et couleurs de cheveux. Pour mieux faire passer le message de ce lancement majeur, les égéries marcheront par deux (blondes, brunes...). On les verra dans tous les médias sans compter les événements en magasins dont un film.
